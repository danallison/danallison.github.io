# The Question Options Acceptance (QOA) Protocol for Group Decision-making

The Question Options Acceptance (QOA) protocol is a formal group decision-making protocol that standardizes the informal process by which small groups typically make decisions naturally. By formalizing this process as a standardized protocol, the process becomes more reliable and can be applied in larger groups across a wider range of contexts. In other words, QOA makes group decision-making scalable.

## The Protocol

1. Ask a question.
2. Make a list of options.
3. Select an option via acceptance voting.
